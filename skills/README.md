# Domain Skills

Domain skills give agents expertise. Each skill is a separate plugin you install alongside the orchestrator.

## Available Skills

| Skill | Plugin | Status |
|-------|--------|--------|
| Content production | `maxtechera/skill-content` | Planned |
| E-commerce operations | `maxtechera/skill-ecommerce` | Planned |
| SEO | `maxtechera/skill-seo` | Planned |
| Sales outreach | `maxtechera/skill-sales-outreach` | Planned |
| Financial reporting | `maxtechera/skill-finance` | Planned |
| Growth / paid acquisition | `maxtechera/skill-growth` | Planned |
| Go-to-market launches | `maxtechera/skill-gtm` | Planned |
| Engineering | Built-in (agent default) | Available |

## Writing Your Own

A skill is a text file. See [docs/VISION.md](../docs/VISION.md) slide 11 for the skill file format.

Install custom skills by dropping them into `~/.orchestrator/skills/`.
