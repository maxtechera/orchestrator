# Kanban Contract

Every ticket must include:

```md
## Inputs
## Deliverables
## Verification
## Artifacts
```

Done gate (all required):

1. State-aware progress comment (`status_summary` + `next_steps`)
2. Verification evidence present
3. Artifacts list updated with canonical links
4. PR URL linked when repo-tracked changes are involved
5. PR merged by human reviewer (agents do not self-merge)
