# Ship launch

Ship is an open source GTM pipeline for Claude Code teams.

## Why it exists
- Build got fast
- Launch stayed manual
- Credential failures still kill momentum

## What it does
1. Validate
2. Build
3. Launch
4. Measure
5. Iterate

## Install
```bash
/plugin marketplace add maxtechera/ship
clawhub install ship
/ship
```
